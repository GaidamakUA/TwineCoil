# TwineCoil
Twine implementation for telegram bot

This implementation aims to support Twine with Harlowe 3.0 story format as well as possible for Telegram chat.

## Features:
### Twine
1. Basic text
2. Links as buttons
3. Images
### Harlowe
1. Hooks
1. Hidden hooks
1. "display" macro
1. "link-reveal" macro
2. "show" macro
